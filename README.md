# YouTube ライブ放送数表示 どんぐりちゃん

## ダウンロード
[ここをクリックしてZipファイルをダウンロード](https://github.com/akaberebere/Youtube-Live-Count-Donguri/archive/refs/heads/main.zip)

## 使い方
1. Zipを解凍します。
2. Chromeで chrome://extensions/ を開きます。
3. 「デベロッパーモード」をONにします。
4. 「展開済み拡張機能を読み込む」から解凍したフォルダを選んでください。





　　　機能説明

   

ブラウザ拡張機能**「YouTube ライブ放送数表示 どんぐりちゃん」**をご利用いただきありがとうございます。

この拡張機能は、お気に入りのYouTubeチャンネルが「今、配信しているか」をあなたの代わりに24時間チェックし、配信が始まった瞬間に音と通知で教えるライブ視聴特化型の通知ツールです。

主な機能を3つのポイントで分かりやすく解説します。

1. 「今」のライブ数が一目でわかる
ブラウザの右上に表示される拡張機能アイコンの上に、現在ライブ配信中のチャンネル数が数字（バッジ）でリアルタイムに表示されます。

　数字が出ているとき: 設定した誰かが配信中です。

　数字がないとき: 誰も配信していません。

クリックすると: 配信中のチャンネル一覧とサムネイルが表示され、クリックですぐに視聴ページへ飛べます。

2. 配信開始を「音」と「通知」でキャッチ
YouTube公式の通知は遅れたり届かなかったりすることがありますが、どんぐりちゃんは最短1分間隔でチェックします。

デスクトップ通知: 画面右下に配信者の名前とサムネイルが表示されます。

カスタム通知音: あなたが設定した好きな音声（MP3など）を鳴らせます。

※「予定」の段階では音は鳴りません。実際に配信が始まった「今」この瞬間にだけ反応します。

3. かんたん設定（予定は自動でスルー）
設定画面（オプション）で、追いかけたいチャンネルの「ハンドル名（@...）」や「検索キーワード」を登録するだけです。

賢いフィルタリング: 「31時間後に配信予定」といった枠（待機所）は自動で除外します。本当に放送が始まっているものだけを教えてくれるので、通知が邪魔になりません。

ハンドル名推奨: @channelname 形式で登録すると、より正確にその人の「ライブ中」を判定できます。

使い方ガイド
登録: 設定画面で好きなチャンネルをリストに追加。

音声設定: 「通知音設定」から、鳴らしたい音声ファイルを保存（空欄なら無音通知になります）。

あとは待つだけ: ブラウザを開いている間、どんぐりちゃんが休まず巡回します。

さらに使いやすくするためのヒント： 特定の「キーワード」で登録しておくと、そのワードに一致するライブが始まった際にも通知を受け取ることができます。
